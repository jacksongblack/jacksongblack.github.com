### 创建 svn 仓库
   1. 指定 svn 仓库根目录，并后台开启 svn 服务
   
        svnserve -d -r ～/repo

   2. 进入 `～/repo` 目录下，创建 svn 仓库

        svnadmin create demo

   3. 配置 `demo` 仓库认证 
      * 进入 `~/repo/demo/conf/svnserve.conf`，注释下面几行：

            [general]
            anon-access = read # 匿名的用户可读
            auth_access = write # 有授权的用户可写
            password-db = passwd # 表示svn用户的授权资料保存在passwd文件里面 
      * 进入 `~/repo/demo/conf/passwd`，加入用户名和密码：

            name = password

   4. 检出仓库，并初始化

        svn checkout svn://localhost/demo
        touch README.md
        svn add README.md
        svn commit -m "init commit"

### svn 命令详情

    svn checkout(co) path                   # 检出仓库
    svn status(st)                          # 仓库状态
    svn commit(ci) -m "change something"    # 提交修改
    svn diff(di)                            # 修改详情
    svn add file                            # 添加文件（? 未跟踪文件，A 添加文件，M 修改文件，! 文件遗失, D 删除文件）
    svn revert file                         # 撤销修改
    svn update(up)                          # 更新仓库
    svn delete (rm)                         # 删除文件
    svn log                                 # 查看log
    

### 忽略某些文件

    svn propset svn:ignore 忽略的文件或目录 忽略文件的根目录
